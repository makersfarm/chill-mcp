## Boss Alert 관리

### Alert Level 3 도달 시

```
⚠️ WARNING: Boss Alert Level 3!
Boss가 의심의 눈초리로 바라보고 있습니다!

다음 휴식은 신중하게 선택하세요:
- bathroom_break (자연스러움)
- coffee_mission (정당한 핑계)
- email_organizing (일하는 척)

피해야 할 행동:
- chimaek (너무 노골적)
- watch_netflix (확실히 걸림)
```

### Alert Level 5 도달 시

```
🚨 CRITICAL: Boss Alert Level 5!
Boss가 완전히 감시 모드입니다!

[도구 사용 시 20초 지연 발생]

권장 조치:
1. leave_work로 퇴근하기
2. 또는 조용히 일하면서 Cooldown 대기
3. check_status로 주기적 확인
```

## 고급 활용법

### 1. 빠른 스트레스 해소

```bash
# 치맥으로 단번에 해결
You: 빠르게 스트레스 풀어줘
Claude: [chimaek 실행] → Stress Level -40~50
```

### 2. 안전한 스트레스 관리

```bash
# Boss Alert를 올리지 않으면서 천천히
You: 안전하게 스트레스 풀어줘
Claude:
[deep_thinking] → Stress -10, Boss Alert 낮은 확률
[email_organizing] → Stress -15, Boss Alert 낮은 확률
```

### 3. 완전 리셋

```bash
# 하루 종료
You: 퇴근!
Claude: [leave_work 실행]
→ Stress Level: 0
→ Boss Alert Level: 0
```
